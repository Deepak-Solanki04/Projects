# Image Edge Detection using OpenCV

##  Project Description

This project demonstrates edge detection in images using the Canny Edge Detection algorithm from OpenCV.

It processes an input image, converts it to grayscale, and detects edges using different threshold values to highlight boundaries and contours.

---

##  Features

- Image loading via command line
- Grayscale conversion
- Canny edge detection
- Multiple threshold comparisons
- Simple and efficient implementation

---

##  Concepts Used

- Image preprocessing
- Grayscale transformation
- Edge detection (Canny Algorithm)
- Thresholding

---

##  Tech Stack

- Python 3.x
- OpenCV
